### 1.2 关键信息提取

#### 1.2.1 开发指南
- **技术栈**：Vue 3 + Vite + Pinia + Vue Router
- **UI组件库**：Element Plus（PC端） + Vant（移动端）
- **项目结构**：三端一体（导购端、门店端、总部端）
- **核心功能**：客户管理、AI智能、营销工具、服务管理、订单管理、供应链管理等
- **开发规范**：代码风格、命名规范、开发流程、分支管理等

#### 1.2.2 设计规范
- **设计风格**：现代简约、专业商务、科技感
- **色彩系统**：品牌主色 #667EEA，辅助色和中性色体系
- **字体系统**：八级标准化字号，Web端使用Inter/思源黑体，移动端使用苹方/思源黑体
- **间距系统**：8px基准栅格，所有间距、内边距、宽高、定位必须为8px的整数倍
- **组件规范**：按钮、输入框、卡片、导航栏等组件的设计标准
- **适配范围**：移动端（320px-768px）、PC端（769px-1920px）

#### 1.2.3 项目架构
- **架构层次**：用户层 → 终端选择层 → 路由层 → 页面层 → 组件层 → 状态管理层 → API层 → 后端服务
- **技术架构**：Vue 3 + Vite + Pinia + Vue Router + Element Plus + Vant + ECharts + Sass
- **目录结构**：清晰的模块化结构，按照终端和功能模块组织
- **核心模块**：认证模块、导购端模块、门店端模块、总部端模块
- **状态管理**：使用Pinia进行状态管理，按照功能模块划分不同的store
- **路由设计**：嵌套路由结构，按照终端和功能模块组织
- **API设计**：使用Axios进行API调用，封装了统一的请求和响应处理
